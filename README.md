# موقع تحويل الـ PDF إلى Word (عربي) 📄✨

**[🔗 جرب الموقع الآن مباشرة!](https://pdf-to-word-arabic-nwzv.vercel.app)**

أداة ذكية مبنية باستخدام الذكاء الاصطناعي (Gemini 2.5 Flash) لاستخراج النصوص العربية من ملفات الـ PDF بدقة متناهية، مع إمكانية تصدير النص كملف Word (DOCX).

## المميزات 🌟
- **نظام حماية ذكي (شاشة قفل 🔒):** الموقع محمي بشاشة قفل تطلب من الزوار إما إدخال كلمة مرور الموقع الخاصة بك، أو استخدام مفتاح API الخاص بهم. هذا يضمن حماية حصتك المجانية من الاستهلاك العشوائي ويسمح لك بمشاركة الموقع بأمان!
- **دقة عالية:** يستخدم قدرات الذكاء الاصطناعي من جوجل (Gemini Vision) لقراءة الملفات كصور، مما يحافظ على صحة النصوص العربية وتنسيقها.
- **تصدير إلى Word:** إمكانية تحميل النص المستخرج كملف `.docx` منسق وجاهز.
- **واجهة عصرية:** تصميم جذاب وسهل الاستخدام (UI/UX) باستخدام Next.js و Tailwind CSS.
- **تفاعلي:** شريط تقدم وتأثيرات بصرية أثناء معالجة الملفات.

## المتطلبات 🛠️
تحتاج إلى تثبيت البرامج التالية على جهازك:
- [Node.js](https://nodejs.org/) (لواجهة المستخدم Frontend)
- [Python 3](https://www.python.org/) (للخادم Backend)
- مفتاح مجاني (API Key) من [Google AI Studio](https://aistudio.google.com/app/apikey)

## طريقة التثبيت والتشغيل 🚀

### 1. إعداد الخادم (Backend - Python)
افتح موجه الأوامر (Terminal) وانتقل لمجلد `backend`:
```bash
cd backend
pip install -r requirements.txt
```

**إعداد المفتاح السري (مهم جداً):**
1. ستجد ملفاً باسم `.env.example`. 
2. قم بنسخه وإعادة تسمية النسخة لتصبح `.env` فقط.
3. افتح ملف `.env` وضع مفتاح جوجل الخاص بك وكلمة مرور موقعك بداخله كالتالي:
   ```env
   GEMINI_API_KEY=ضع_مفتاحك_هنا
   SITE_PASSWORD=كلمة_المرور_التي_تريدها
   ```

**تشغيل السيرفر:**
```bash
uvicorn main:app --reload
```

### 2. إعداد واجهة المستخدم (Frontend - Next.js)
افتح نافذة أوامر جديدة (Terminal) وانتقل لمجلد `frontend`:
```bash
cd frontend
npm install
npm run dev
```

الآن افتح المتصفح على الرابط `http://localhost:3000` وابدأ برفع ملفاتك!
